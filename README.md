# 驻马店市政府工作报告文本分析

爬取并分析驻马店市历年（2002-2026）政府工作报告，探索政策关注点的演变趋势。

## 项目结构

```
├── crawl.py         # 爬虫：从政府网站抓取历年报告
├── analyze.py       # 分析脚本：生成趋势图表和词云
├── data/            # 爬取到的报告文本数据
├── output/          # 分析输出的图表（运行后生成）
└── requirements.txt # Python 依赖
```

## 使用方法

```bash
pip install -r requirements.txt

python crawl.py      # 爬取最新报告
python analyze.py    # 分析并生成图表
```

## 分析维度

- **政策关注度趋势**：AI/科技、环保/绿色、民生/社保、经济发展四大领域
- **文本一致性**：相邻年份报告的余弦相似度
- **基础统计**：篇幅变化、句长变化、词汇丰富度
- **政策转变对比**：2014年前后关注点雷达图
- **情感分析**：基于 SnowNLP 的句子级情感倾向
- **词云**：TF-IDF 加权关键词可视化
